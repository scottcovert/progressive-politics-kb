## Data Format

Every entry includes:

```json
{
  "source": "Original source (video, research paper, report)",
  "sourceType": "video-summary | research-synthesis | data-compilation",
  "category": "One of 16 categories",
  "summary": "What this entry covers",
  "keyFacts": ["Up to 8 sourced facts per entry"],
  "biasFlags": ["Noted when source has ideological lean"],
  "tags": ["Searchable tags"],
  "url": "Source URL when available"
}
```

**Bias transparency:** Entries sourced from advocacy channels are flagged. The goal is ammunition that holds up to scrutiny, not propaganda.
